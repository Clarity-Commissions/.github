# Clarity Commissions

**Infrastructure for commission reconciliation.**

Clarity Commissions is a private SaaS platform for ingesting commission statements, normalizing messy real-world data, applying payout policies, and surfacing discrepancies (missed payouts, incorrect tiers, and other “silent leaks”).

## What it does
- **Statement ingestion**: Import commission statements from multiple sources and formats.
- **Normalization**: Standardize schemas, map headers, and handle inconsistent layouts.
- **Policy rules engine**: Model payout logic (tiers, effective dates, plan types, renewals vs. new business, overrides).
- **Reconciliation & audit**: Compare “expected vs. paid” and generate evidence-backed exception reports.
- **Config-driven onboarding**: Templates and mappings to support multiple carriers/partners and workflows.
